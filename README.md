# CODA-19: CSV to FHIR

This repository contains scripts to generate FHIR resources from data in a tabular / relational format.

## Requirements

Python v3

## Install

In order to install, clone the repository:

```
git clone https://github.com/CODA-19/csv-to-fhir
cd csv-to-fhir
```
